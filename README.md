# Proyecto Final Springboot - SysOne University

- Santiago Fagliano

## Enunciado

Proyecto sobre un ABM de la liga de equipos de la AFA. Ademas cuenta con distintas capas de Spring Boot:

- Capa modelo:
  @Entitie (JPA/Hibernate)
- Capa DAO:
  @Repositories (spring)
- Capa Service:
  @Service (spring)
- Capa controlador:
  @RestController (spring)
